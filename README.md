## 项目简介

基于阿里巴巴天池 UserBehavior 真实数据集，完成电商平台用户从「浏览」到「购买」的全链路行为分析，识别核心增长机会，输出可落地的分层运营策略。

## 数据来源

- **数据集**：阿里巴巴天池 [UserBehavior Dataset](https://tianchi.aliyun.com/dataset/649)
- **规模**：1亿+ 行为事件，约 98 万真实用户
- **时间窗口**：2017年11月25日 – 12月3日（覆盖双十二大促预热期）
- **行为类型**：浏览(pv)、加购(cart)、收藏(fav)、购买(buy)

## 分析模块

| 模块 | 核心结论 |
|------|---------|
| 用户行为漏斗 | PV→购买整体转化率 2.25%，PV→加购是最大流失节点（流失 93.8%） |
| 日度趋势分析 | 双十二预热日(12月2日) PV 提升 35.1%，加购提前于购买爆发 |
| 用户活跃时段 | 双峰结构：上午 10–11 时 & 晚间 21–22 时为购买高峰 |
| 用户分层 | 四层分级，「活跃未转化」用户（8.9万人）为最高优先运营对象 |
| 运营策略建议 | 四项可落地策略，含 A/B 测试因果推断方法论 |

## 技术栈

- **数据处理**：Python · pandas（流式处理 3.5GB 原始数据）
- **可视化**：Chart.js（交互式漏斗图、趋势折线图、时段热力图、用户分层饼图）
- **呈现形式**：单文件自包含 HTML，离线可用
